# Starcode – Full-Stack Web & App Development

Starcode is a premium software development agency specializing in high-performance web applications, scalable mobile apps, and intelligent business automation solutions.

## Our Expertise

- **Web Development**: Next.js, React, Node.js, and TypeScript.
- **Mobile Apps**: Flutter and React Native for cross-platform excellence.
- **Business Automation**: Custom workflows and system integrations.
- **UI/UX Design**: Modern, user-centric interfaces.

## Technology Stack

- **Frontend**: Next.js 15, React 19, Tailwind CSS, GSAP, Swiper.
- **Backend**: Supabase, PostgreSQL.
- **Languages**: TypeScript, JavaScript.

## Getting Started

First, install dependencies:

```bash
npm install
```

Then, run the development server:

```bash
npm run dev
```

Open [http://localhost:3000](http://localhost:3000) with your browser to see the result.

## Build and Deployment

To create an optimized production build:

```bash
npm run build
```

To start the production server:

```bash
npm run start
```
